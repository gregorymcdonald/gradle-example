# **Gradle Example**
**Author**: Greg McDonald  

## Run Instructions
**Run command**:  
`gradle assemble`  
OR  
`gradle build` (run `build` if you want to run tests)

**Then run command**:  
`java -jar build/libs/gradle-example.jar`

**Reference**: https://www.petrikainulainen.net/getting-started-with-gradle/
